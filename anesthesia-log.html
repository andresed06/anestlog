Backup & Acceso
Protege tus datos — exporta regularmente

📱 Cómo instalar en tu celular como App

Android (Chrome): Abre el archivo HTML → menú ⋮ → "Agregar a pantalla de inicio" → queda como ícono de app
iPhone / iPad (Safari): Abre el archivo en Safari → ícono □↑ compartir → "Agregar a pantalla de inicio"

💡 Acceso desde cualquier dispositivo: Guarda el archivo anesthesia-log.html en Google Drive o iCloud. Ábrelo desde el navegador del celular y agréga a pantalla de inicio. Todos tus datos quedan en ese dispositivo.

💾 Exportar datos

CSV: Abre en Excel, Google Sheets, o SPSS para análisis estadístico.
JSON: Backup completo con todos los campos. Guárdalo en la nube cada semana.

📂 Importar / Restaurar backup

Selecciona un archivo JSON exportado previamente para restaurar todos los casos.
⚠ Esto reemplaza los datos actuales. Haz un backup antes de importar.

🔒 Recomendaciones de seguridad

• Exporta JSON cada semana y guárdalo en Google Drive, iCloud o Dropbox.
• No borres el caché del navegador sin hacer backup primero — los datos se almacenan en localStorage.
• Guarda el archivo .html junto con tus backups JSON en la misma carpeta de nube.
• Los datos son 100% privados — se almacenan solo en tu dispositivo, nunca se envían a ningún servidor.
• Futuro: versión con sincronización automática a Google Sheets disponible próximamente.

⚠ Zona de peligro

Esta acción es irreversible. Exporta un backup antes de proceder.
